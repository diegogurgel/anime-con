### Windows  
```shell
	npm install -g ionic
``` 

### Mac and Linux. 
```shell
	sudo npm install -g ionic
``` 

### Start the app. 
```shell
	ionic start anime-con -t blank
```
### If isn't present add the android platform  
```shell
	ionic platform add android 
```
